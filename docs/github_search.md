1. `in:name xxx // 按照项目名搜索`
2. `in:readme xxx // 按照README搜索`
3. `in:description xxx // 按照description搜索`



1. `stars:>xxx // stars数大于xxx`
2. `forks:>3000 // forks数大于xxx`
3. `language:xxx // 编程语言是xxx`
4. `pushed:>YYYY-MM-DD // 最后更新时间大于YYYY-MM-DD`
